关于学习求职等的一点点牢骚

发表于 2017-09-18 | 分类于 生活

最近很多企业来学校招聘，有名一点的今天的中国移动、明天的中国联通，我忍不住想明年暑假的我自己，我一直在想，企业根本不会因为我一点点都不通的切图伎俩而要我，要招的话，同等条件下，肯定都选择一个看着聪明的，我越想越觉得害怕，可是有什么用呢，我好像越写越过分。

我不知道其他想要就业的人怎么想，总之我的心里一直萦绕着这件事，可是每当我鼓足信心，或者因为看到一些什么事一猛子扎下去学的时候，效果一点都不理想，到了第二天，连这种一猛子的冲动都没有了，我不知道这种心态的产生是因为什么，说实话，我是想对学习充满热情的，我在网上找怎样学习，怎样学习效果好，等等这些东西，我觉得我的心，我的情绪越来越不属于我自己，我想我自己变得随性一点，再随性一点，等对一些事情看的淡一点，毕竟都没有什么特别让人难以释怀的。

我作为一个自认为想要当前端的人，竟然害怕切图，我心里害怕这个东西，我想了想我害怕的原因，是因为做的不好，我想起很多很多次xz，我觉得他做的那么好，我也想跟他一样，很想很想有一天告诉他，我也跟他是同事了，这该是多么幸福的事情，对啊，这是我做梦都想要实现的事情。

回归一下积极的方面，既然还没有放弃，也没有打算放弃何不破釜沉舟，认认真真的做，就这一年的时间，成功成仁就是这样了，总之，该学还是要学，不过是要聪明的学，牢记：学技术是要去解决问题，不论是什么东西，该放下还是放下，积极一点，如果要做，那就好好做，不要破坏这个行业的尊严。

​吾辈当自强。
